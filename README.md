# Sugirun Game

障害物に当たらないように走り続けるラン系ゲームです。

## 概要

プレイヤーを操作し、出現する障害物を避けながら長く生存する（または得点を稼ぐ）ことを目指します。

### プロトタイプの仕様
- プレイヤーは左右移動とジャンプのみ
- 障害物は前方から流れてくる
- 3回ぶつかったらゲームオーバー
- スコアは生存時間で加算
